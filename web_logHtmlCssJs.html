Some Elements in HTML5

  • <article></article>
  • <xmp></xmp> (used to translate code definition. A lagacy element. The recommended elements now are <pre></pre> and <code></code>)
  • <aside></aside>
  • <details></details> (gives additional details that the user may use or hide)
  • <dialog></dialog>
  • <figcaption></figcaption>
  • <figure></figure>(defines content like illustrations, photos, diagrams, code listing etc)
  • <footer></footer> (defines a footer for a section or the document)
  • <header></header> (describes the header for the document)
  • <main></main> (main content of the document is defined)
  • <mark></mark> (shows highlighted or marked text)
  • <menuitem></menuitem> (an item from the menu which user can use from the popup menu bar)
  • <meter></meter> (measurement within a known range)
  • <nav></nav> (shows navigation links in the document)
  • <progress></progress> (shows the progress of a task)
  • <rp></rp> (describes what to show in the browser if the browser do not support "Ruby annotation". It is used together with <rubi></rubi> and <rt></rt> elements)
  • <section></section>
  • <summary></summary> (visible heading for a "detail" element)
  • <time></time> (give date and time)
  • <wbr> (word break opportunity)
  • <br> (line break)
  • <bdi></bdi> (isolates a span of text from its surrounding text for bidirectional formatting. The browser's bidirectional algorithm determines the text direction It's particularly useful when embedding text with an unknown directionality, like when incorporating Arabic or Hebrew text into a predominantly left-to-right context.)

Attributes in HTML5

  • align
    options (left, right, center)
    function (aligns the tag horizontally)
  • bgcolor
    options (hexadecimals, numeric, RGB values)
    function (a background color is placed behind an element)
  • accesskey
    options (user defined)
    function(defines a keyboard shortcut to access an element)
  • class
  • height
    options (numeric value)
    functions (defines the height of images, tables por table cells)
  • item
    options(list of elements)
    function (is used to group elements)
  • spellcheck
    option ( true or false )
    function (specifies if element requires spelling for grammar check)
  • width
    option (numeric value)
    function (defines the width of images, tables por table cells)
  • title
    option (user defined)
    function (pop-up title for elements)

Some Interesting features of HTML5

  • Multimedia Support in HTML5 <pre><code> <figure>Syntax to play video: <body><video src="video path" controls></video></body> </figure> <figure>Syntax to play audio: <body> <audio src="audio location" controls></audio></body> </figure> </code></pre>
  • New Input Element Types <pre><code> Example of existing "input" are "file", "password" Example of new "input" element is "email" <figure><input id="EmailIdInput" type="email"/></figure> </code></pre>
  • Canvas:
    It is a rectangular area that allows pixel level operations such as drawing a box, line, circle, performing graphics etc, Sample syntax: <pre><code> <canvas id="mybox" width="50" height="50"> </code></pre>
  • Custom Data Attributes: they can be added in a hidden manner from the user and can be used later by js functions. These attributes need to start with "data-" <pre><code> Example: <br> <div class="example" data-subject="maths" data-level="complex"></div> </code></pre>
  • Editable Contents: The end users can edit HTML control's content under this feature. It also allows developers to create web pages with sections like HTML editor, notes etc.
    In order to use this feature add following to the control panel of HTML: contentEditable="true"
  • Range Validators and Required "Required" syntax <pre><code> <input type="text" name="TextBox" required/> </code></pre>
  • "Range validator" syntax <pre><code> <input type="number" name="Number of Items" min="5" max="10" required/> </code></pre>
  • Placeholder and Autofocus Attributes syntax: <pre><code> <button id="submitbutton" autofocus></button> <input type="text" name="text box" placeholder="Please enter text..."/> </code></pre>
  • Preload Videos: This attribute enables the browser to preload the video.
  • Geo-Location: This feature of html5 let you share your location with websites of your choice. It works well on smart phones..
  • Figure Element By using this feature one can semantically associate image with their captions. We simply need to add <figcaption> element. Below is a sample of this element: <pre><code> <figure> <img src="path/to/image" alt="About Image"/> <figcaption><p>This is a beautiful scenery.</p></figcaption></figure> </code></pre>
